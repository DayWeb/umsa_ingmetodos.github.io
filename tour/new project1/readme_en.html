Description:

For security reasons, browser doesn't allow users to load projects containing flash or webGL locally. You must upload the projects to web server and view the project by visiting http URL. Easypano offers a free website for uploading works from users who don't have own server. You could upload panorama images or virtual tours to Ep-Sky or synchronize the project directly to EP-Sky server through the synchronization function of the VRTourMaker publishing interface.

Browser Requirement:

Based on webGL technology, the virtual tours require IE 10.0 or higher, Safari 5.0 or higher, Chrome 21.0 or higher when viewed on a PC or Mac. Viewed on iPhone, it ,need iPhone 4 or above and it should Android 5.0 or higher when viewed on an Android device.
